# Topology: layered DAG, all edges strictly increase vertex index
# Why easy: edges only flow forward (u → v where u < v), so by the
# time v is dequeued, d[u] for all predecessors u is already final.
# No node's estimate can improve after first processing.
# ─────────────────────────────────────────────────────────────
